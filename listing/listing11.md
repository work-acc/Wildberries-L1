Что выведет данная программа и почему?

```go
func main() {
  wg := sync.WaitGroup{}
  for i := 0; i < 5; i++ {
     wg.Add(1)
     go func(wg sync.WaitGroup, i int) {
        fmt.Println(i)
        wg.Done()
     }(wg, i)
  }
  wg.Wait()
  fmt.Println("exit")
}

```

Ответ:
```
Данная программа выведет последовательность чисел от 0 до 4 
и затем строку "exit". Последовательность чисел может быть 
неупорядоченной из-за выполнения горутин параллельно, 
но обязательно будут выведены все числа от 0 до 4.

Однако программа не завершится сразу после вывода чисел. 
Причина в том, что аргумент wg передается в анонимную 
функцию по значению, а не по ссылке. Это означает, 
что каждая горутина работает с копией sync.WaitGroup, 
и вызов метода wg.Done() увеличит счетчик ожидания 
для этой копии, но не для оригинальной sync.WaitGroup. 
Поэтому программа "зависнет" на строке wg.Wait(), 
так как оригинальная sync.WaitGroup не будет ожидать 
завершения горутин.
```
