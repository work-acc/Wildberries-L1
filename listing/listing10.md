Что выведет данная программа и почему?

```go
func update(p *int) {
  b := 2
  p = &b
}

func main() {
  var (
     a = 1
     p = &a
  )
  fmt.Println(*p)
  update(p)
  fmt.Println(*p)
}

```

Ответ:
```
Output:
 1
 1

Причина такого вывода заключается в том, что функция update 
принимает указатель p на int, но при вызове функции update(p) 
она присваивает указателю p новое значение - адрес переменной b. 
Однако, это изменение применяется только внутри функции update, 
и когда функция завершается, это изменение теряется, и p снова 
начинает указывать на переменную a. Поэтому после вызова функции 
update(p) значение, на которое указывает p, остается неизменным, 
и программа выводит 1 дважды.
```
