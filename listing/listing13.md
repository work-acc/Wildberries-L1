Что выведет данная программа и почему?

```go
func someAction(v []int8, b int8) {
  v[0] = 100
  v = append(v, b)
}

func main() {
  var a = []int8{1, 2, 3, 4, 5}
  someAction(a, 6)
  fmt.Println(a)
}

```

Ответ:
```
Output: [100 2 3 4 5]

Это происходит потому, что срезы в Go являются ссылочными 
типами данных, и при передаче среза в функцию передается 
копия ссылки на оригинальный массив, а не копия самого 
массива. Поэтому, когда вы изменяете элементы среза 
внутри функции someAction, эти изменения отражаются 
на оригинальном срезе a.

Однако, когда вы добавляете новый элемент в срез 
с помощью функции append, создается новый срез, 
который уже не связан с оригинальным срезом a. 
Поэтому изменения, внесенные в v после append, 
не влияют на оригинальный срез a.

В результате, элемент a[0] изменяется на значение 
100, но добавление элемента b в v не отражается 
на оригинальном срезе a.
```
