В чем разница make и new?

Ответ:
```
Основное различие между make и new заключается в том, 
что make используется для создания ссылочных типов данных 
(карты, срезы, каналы), в то время как new используется 
для выделения памяти для создания нового экземпляра любого 
типа данных и возвращает указатель на этот экземпляр.
```
