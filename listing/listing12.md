Что выведет данная программа и почему?

```go
func main() {
  n := 0
  if true {
     n := 1
     n++
  }
  fmt.Println(n)
}

```

Ответ:
```
Output:
0

Данная программа выведет 0.
Это происходит из-за того, что внутри блока if true 
объявляется новая переменная n с помощью оператора :=, 
которая имеет область видимости только внутри этого блока.
 Это означает, что внешняя переменная n, объявленная 
 в начале функции, не изменяется. Поэтому после выполнения 
 блока if значение внешней переменной n остается равным 0, 
 и программа выводит 0.
```
